In Python, variables can store different types of data. Each type has its own characteristics and operations that can be performed on it.

Common Data Types

1

INTEGERS (int)

Whole numbers without decimal points.

age = 25

Operations: Addition, subtraction, multiplication, division, etc.

2

FLOATING POINT (float)

Numbers with decimal points.

price = 19.99

Operations: Same as integers, plus precise decimal calculations.

3

STRINGS (str)

Text enclosed in quotes.

name = "Alice"

Operations: Concatenation, slicing, methods like upper(), lower(), etc.

4

BOOLEANS (bool)

Either True or False.

is_active = True

Operations: Logical operations (and, or, not) and comparisons.

5

LISTS

Ordered collections of items.

scores = [95, 87, 92]

Operations: Indexing, slicing, appending, sorting, etc.

6

OTHER DATA TYPES

  • Dictionaries (dict): Key-value pairs
  • Tuples: Immutable ordered collections
  • Sets: Unordered collections of unique items
  • None: Represents the absence of a value

Checking Variable Types

You can discover a variable's type using the type() function:

age = 25 name = "Alice" price = 19.99 print(type(age)) # Output: print(type(name)) # Output: print(type(price)) # Output:

Python is a dynamically typed language, which means variable types are determined automatically based on the assigned value.

2

Watch Out!

Type Conversion Errors

Be careful when mixing different types in operations!

"age: " + 25 # Error! Can't add string and integer
"age: " + str(25) # Correct! Convert int to string first

Don't Forget String Quotes

Strings must always be enclosed in quotes!

name = Alice # Error! Python thinks Alice is a variable
name = "Alice" # Correct! Use quotes for text

Compound Assignment Operators

These operators perform an operation and assign the result back to the variable in one step:

Numeric Examples

x = 10 x += 5 # Same as: x = x + 5 (x is now 15) x -= 3 # Same as: x = x - 3 (x is now 12) x *= 2 # Same as: x = x * 2 (x is now 24) x /= 4 # Same as: x = x / 4 (x is now 6.0)

These operators are shortcuts that make your code more concise.

String Examples

message = "Hello" # Add to the existing string message += " there" # message is now "Hello there" message += "!" # message is now "Hello there!"

The += operator is commonly used to build strings incrementally.

Step 1: Choose a Variable Name

First, choose a descriptive name for your variable. This name will be used to reference the data later in your code.

Naming Rules:

  • Start with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Be descriptive about what the variable contains
  • Case-sensitive (score and Score are different)

Don't:

  • Start with a number (e.g., 1score)
  • Use special characters (@, #, $, etc.)
  • Use Python keywords (if, for, while, etc.)
  • Use spaces in names
player_score
Variable Name

This descriptive name helps you remember what value is stored and makes your code readable. Choose names that explain what the variable represents.

Step 2: Add the Assignment Operator

Next, add the assignment operator (=) after the variable name. This tells Python you're about to assign a value to this variable.

  • The equals sign (=) is the assignment operator in Python
  • It assigns the value on its right to the variable on its left
  • Don't confuse it with the equality comparison operator (==)
player_score
=
Assignment Operator

The equals sign (=) is the assignment operator. It tells Python to take the value on the right and store it in the variable on the left.

Step 3: Assign a Value

Finally, add the value you want to store in the variable. This can be a number, string, boolean, or other Python data type.

Common Value Types:

Integers
score = 42
Strings
name = "Alice"
Floats
price = 19.99
Booleans
is_valid = True
player_score =
100
Value

The value (100) is what gets stored in the variable. Python automatically determines the appropriate data type based on the value (in this case, an integer).

Step 4: The Complete Variable Declaration

Let's put it all together! The complete variable declaration statement creates a new variable with your chosen name and assigns it a value.

What Happens When This Code Runs:

  1. Python creates a new variable named player_score
  2. It allocates memory to store the integer value 100
  3. It associates the variable name with that memory location
  4. Now you can use player_score elsewhere in your code
player_score = 100
Complete Statement

This complete statement creates a variable named player_score and assigns it the integer value 100. After this line executes, you can use player_score in your program.

Changing Variable Values

Once a variable is created, you can change its value later in your program.

Step 1: Initial Variable Declaration

First, we create a variable with an initial value:

player_score = 100
# The variable 'player_score' now holds the value 100
Initial Value

When we first create the variable, we assign it an initial value. This is the starting point.

Step 2: Reassigning a New Value

Later in the program, we can assign a completely new value to the same variable:

player_score = 100 # Some game code happens here...
player_score = 150
New Value Assignment

The same variable name now refers to a new value (150). The old value (100) is no longer accessible through this variable.

Step 3: Calculating New Value

Often, we calculate the new value based on the current value:

player_score = 100 # Player earns 50 more points
player_score = player_score + 50
Calculation-Based Update

This takes the current value of player_score (100), adds 50 to it, and then stores the result (150) back in the player_score variable.

Step 4: Using Shorthand Operators

Python provides shorthand operators for common operations:

player_score = 100 # These two lines are equivalent: player_score = player_score + 50
player_score += 50
Shorthand Operators

The += operator is shorthand for "add and assign." Similar operators include -=, *=, /=, etc. These make your code more concise.

Step 1 of 4
Multiple Variables

Working with Multiple Variables

Python provides several ways to work with multiple variables efficiently.

Step 1: Multiple Assignment

You can assign values to multiple variables in a single line:

x, y, z = 10, 20, 30
# You can check each value print(x) # 10 print(y) # 20 print(z) # 30
Multiple Assignment

This assigns 10 to x, 20 to y, and 30 to z all in one line. The number of variables must match the number of values.

Step 2: Same Value to Multiple Variables

You can assign the same value to multiple variables:

width = height = depth = 10
# All variables have the same value print(width) # 10 print(height) # 10 print(depth) # 10
Same Value Assignment

This creates three variables (width, height, and depth) and assigns the value 10 to all of them. This is useful for initializing related variables.

Step 3: Swapping Variables

Python makes it easy to swap variable values:

# Initialize variables a = 5 b = 10 # In other languages, you need a temporary variable: # temp = a # a = b # b = temp # In Python, you can swap in one line:
a, b = b, a
print(a) # 10 print(b) # 5
Variable Swapping

This elegant syntax swaps the values of a and b. Python handles the temporary storage internally, making your code cleaner.

Step 1 of 3

Variable Naming Rules

Valid Variable Names:

  • score - Simple, one word
  • player_score - Words separated by underscore
  • playerScore - Camel case (second word capitalized)
  • score1 - Letters and numbers
  • _hidden - Can start with underscore

Invalid Variable Names:

  • 123score - Cannot start with a number
  • player-score - Cannot use hyphens
  • if - Cannot use reserved keywords
  • player score - Cannot contain spaces
  • player@score - Cannot use special characters

Variable Naming Best Practices

1

BE DESCRIPTIVE

x = 28 (Unclear what this represents)
age = 28 (Clear meaning)
2

USE CONSISTENT STYLE

Snake Case (Python's preference):

player_score = 100

Camel Case (Alternative):

playerScore = 100
3

AVOID RESERVED WORDS

Don't use Python keywords as variable names:

if = 5 (Error: 'if' is a Python keyword)

Other reserved words: for, while, class, return, True, None, etc.

4

LENGTH AND READABILITY

a = 3.14159 (Too short)
the_mathematical_constant_pi = 3.14159 (Too long)
pi_value = 3.14159 (Just right)
3

Common Variable Mistakes

Forgetting to Initialize

Variables must be assigned a value before they're used!

print(score) # Error if score hasn't been created
score = 0 print(score) # Correct: initialize first

Case Sensitivity

Variable names are case-sensitive!

name = "Alice" print(Name) # Error: Name is not the same as name
name = "Alice" print(name) # Correct: same case

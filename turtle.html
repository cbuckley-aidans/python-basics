Introducing the Python Turtle Module

The Python Turtle module lets you control a virtual "turtle" that moves and draws on a screen.

1

Getting Started

To use Turtle, we first need to import it:

from turtle import *

This gives us access to all the turtle commands without needing to type turtle. before each one.

forward(100) # Move the turtle forward 100 pixels
2

Basic Turtle Commands

MOVEMENT COMMANDS

forward(100) # Move 100 pixels forward backward(50) # Move 50 pixels backward right(90) # Turn right 90 degrees left(45) # Turn left 45 degrees

PEN COMMANDS

penup() # Stop drawing when moving pendown() # Start drawing when moving write("Text") # Write text at the current position
3

Position and Speed

POSITIONING

goto(-100, 50) # Move to x=-100, y=50 # The center of the screen is (0,0) # Default screen is 400x400 pixels

SPEED CONTROL

speed(10) # Fast speed speed(1) # Slow speed speed(0) # Fastest speed (no animation)

Simple Turtle Example

# A simple turtle program that draws a line from turtle import * forward(100) # Move forward 100 pixels right(90) # Turn right 90 degrees forward(50) # Move forward 50 pixels

Key Point: The turtle starts in the middle of the screen (coordinates 0,0) pointing to the right.

Creating a Race Track with Turtle

Now we'll use turtle commands to draw a race track with numbered lanes.

1

The write() Function

The write() function displays text at the turtle's position:

from turtle import * write(0) # Write the number 0 forward(20) # Move forward 20 pixels write(1) # Write the number 1
# Write with alignment options write(5, align='center') # Center-aligned text
2

Drawing Track Lines

We can use a loop to draw evenly spaced track lines:

Track Drawing Code Breakdown Step 1 of 5
from turtle import * speed(10) # Speed up the turtle penup() goto(-140, 140) # Position in top-left

Step 1: Set up the turtle

First we set the speed and position the turtle in the top-left

We use penup() so it doesn't draw while moving to the start position

from turtle import * speed(10) penup() goto(-140, 140) for step in range(6): write(step, align='center')

Step 2: Write track numbers

We create a loop that runs 6 times (0 through 5)

The write() function displays each number

from turtle import * speed(10) penup() goto(-140, 140) for step in range(6): write(step, align='center') right(90) # Turn right 90 degrees forward(10) # Move forward a bit

Step 3: Prepare to draw lines

After writing each number, the turtle turns right 90 degrees

Then moves forward 10 pixels to leave a small gap before the line

from turtle import * speed(10) penup() goto(-140, 140) for step in range(6): write(step, align='center') right(90) forward(10) pendown() # Start drawing forward(150) # Draw a vertical line penup() # Stop drawing

Step 4: Draw track lines

Now we put the pen down to start drawing

Draw a vertical line 150 pixels long

Then lift the pen up again

from turtle import * speed(10) penup() goto(-140, 140) for step in range(6): write(step, align='center') right(90) forward(10) pendown() forward(150) penup() backward(160) # Move back to the top left(90) # Turn left to face right again forward(20) # Move right for the next line

Step 5: Reposition for next line

Move the turtle back to the top with backward(160)

Turn left 90 degrees to face right again

Move 20 pixels to the right to position for the next line

3

Extending the Track

To create a longer track with 16 lines instead of 6, we just need to change one value:

# Change from range(6) to range(16) for step in range(16): write(step, align='center') # rest of the code unchanged

Key Point: This will create 16 vertical lines numbered from 0 to 15 at the top.

Complete Track Code

from turtle import * speed(10) penup() goto(-140, 140) for step in range(16): write(step, align='center') right(90) forward(10) pendown() forward(150) penup() backward(160) left(90) forward(20)

Creating Multiple Turtles for Racing

Now let's create multiple turtle objects for a racing game!

1

Creating a Turtle Object

We can create a new turtle with a name using the Turtle() class:

from turtle import * # Create a new turtle named 'ada' ada = Turtle() ada.color('red') # Set the turtle's color ada.shape('turtle') # Change shape to look like a turtle

Key Point: Each turtle is independent and can be controlled separately using dot notation (e.g., ada.forward(50)).

2

Positioning Turtles at Starting Line

We'll place each turtle at a different y-coordinate, but the same x-coordinate:

# Position the first turtle at starting line ada.penup() ada.goto(-160, 100) # x = -160, y = 100 ada.pendown() # Position the second turtle below the first bob.penup() bob.goto(-160, 70) # Same x, different y bob.pendown()

Tip: Position the turtles with enough space between them (30 pixels here) so they don't overlap during the race.

3

Making Turtles Race

We'll use the random module to make our turtles move a random distance each turn:

from random import randint # Import random integer function # Race loop - each turtle moves in each turn for turn in range(100): # 100 turns in the race ada.forward(randint(1, 5)) # Move 1-5 pixels randomly bob.forward(randint(1, 5))
Understanding randint()

The randint() Function:

  • randint(a, b) returns a random integer between a and b (inclusive)
  • In our example, randint(1, 5) will return either 1, 2, 3, 4, or 5 randomly
  • Each time the function is called, it returns a different random number
  • This makes each turtle move a different, unpredictable distance each turn

Complete Racing Program

Here's how all the pieces come together to create a complete turtle racing program:

from turtle import * from random import randint # Set up screen and drawing speed speed(0) # Draw race track penup() goto(-140, 140) for step in range(16): write(step, align='center') right(90) forward(10) pendown() forward(150) penup() backward(160) left(90) forward(20) # Create first turtle (red) ada = Turtle() ada.color('red') ada.shape('turtle') ada.penup() ada.goto(-160, 100) ada.pendown() # Create second turtle (blue) bob = Turtle() bob.color('blue') bob.shape('turtle') bob.penup() bob.goto(-160, 70) bob.pendown() # Race the turtles for turn in range(100): ada.forward(randint(1, 5)) bob.forward(randint(1, 5))

Variables

Variables are containers for storing data values. In Python, you don't need to declare a variable type - it's determined by the value assigned.

Syntax Breakdown:

variable_name
Choose a descriptive name using letters, numbers, and underscores (cannot start with a number)
=
Assignment operator - stores the value on the right into the variable on the left
value
The data you want to store (number, text, boolean, etc.)

Basic Data Types:

Strings (text)

name = "Alice"
message = 'Hello, world!'

Text enclosed in quotes (single or double)

Integers (whole numbers)

age = 12
count = -7

Whole numbers without a decimal point

Floats (decimal numbers)

height = 1.65
temperature = 21.5

Numbers with a decimal point

Booleans (True/False)

is_student = True
has_completed = False

Special values: either True or False

Lists (collections)

fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]

Ordered collections of items

Common Errors

Type Mismatch: Trying to do math with strings

age = "13"  # This is a string, not a number
next_year = age + 1  # Error: Can't add a number to a string

‚ùå Error: TypeError: can't concatenate str and int objects

‚úÖ Fix: Convert the string to a number first: age = int("13")

Invalid variable names

1name = "Alice"  # Cannot start with a number
my-age = 12  # Cannot use hyphens

‚ùå Error: SyntaxError: invalid syntax

‚úÖ Fix: Use valid names like name1 and my_age

print() Function

The print() function displays text or values to the screen. It's one of the most basic and useful functions in Python!

Syntax Breakdown:

print(
The function name followed by an opening parenthesis
"Hello, World!"
The value you want to display (text requires quotes)
)
Closing parenthesis

Common Errors

Forgetting quotes around text

print(Hello, World!)

‚ùå Error: NameError: name 'Hello' is not defined

‚úÖ Fix: print("Hello, World!")

Mismatched or missing parentheses

print("Hello, World!"

‚ùå Error: SyntaxError: unexpected EOF while parsing

‚úÖ Fix: print("Hello, World!")

Mismatched quotes

print("Hello, World!')

‚ùå Error: SyntaxError: EOL while scanning string literal

‚úÖ Fix: print("Hello, World!") or print('Hello, World!')

input() Function

The input() function allows your program to get information from the user.

Syntax Breakdown:

variable =
The name where you'll store the user's input
input(
The input function with opening parenthesis
"Enter your name: "
A prompt to tell the user what to enter (optional)
)
Closing parenthesis

Important: input() always returns a string, even if the user types a number.

To convert the input to a number, use:

age = int(input("How old are you? "))  # Convert to integer
height = float(input("How tall are you? "))  # Convert to decimal

Common Errors

Trying to do math with string inputs

age = input("How old are you? ")  # Returns a string
next_year = age + 1  # Error: Can't add string and number

‚ùå Error: TypeError: can't concatenate str and int objects

‚úÖ Fix: age = int(input("How old are you? "))

Forgetting to store the input in a variable

input("What's your name? ")
print("Hello,", name)  # Error: 'name' is not defined

‚ùå Error: NameError: name 'name' is not defined

‚úÖ Fix: name = input("What's your name? ")

if/elif/else Statements

If statements let your program make decisions based on conditions.

Syntax Breakdown:

if condition
Test if this condition is True
:
Required colon marks the beginning of a code block
code block
Indented code that runs if the condition is True (use the tab key)
elif another_condition
"else if" - checks another condition if the first was False
:
Required colon marks the beginning of a code block
code block
Runs if this condition is True (use the tab key)
else
Runs if all conditions above were False
:
Required colon marks the beginning of a code block
code block
Code that runs if all conditions were False (use the tab key)

Common Errors

Forgetting the colon (:)

    print("It's hot today!")

‚ùå Error: SyntaxError: invalid syntax

‚úÖ Fix: if temperature > 30:

Incorrect indentation

print("It's hot today!")

‚ùå Error: IndentationError: expected an indented block

‚úÖ Fix: Indent the code inside the if statement with the tab key

Using = instead of == for comparison

    print("It's exactly 30 degrees!")

‚ùå Error: SyntaxError: invalid syntax

‚úÖ Fix: if temperature == 30:

Booleans

Booleans are simple values that can only be True or False. They're used in conditions and comparisons.

Common Boolean Operators:

Comparison Operators:

  • == Equal to
  • != Not equal to
  • < Less than
  • > Greater than
  • <= Less than or equal to
  • >= Greater than or equal to

Logical Operators:

  • and Both conditions must be True
  • or At least one condition must be True
  • not Inverts a boolean (True becomes False)

Common Errors

Using = instead of == for comparison

    print("x is True")

‚ùå Error: SyntaxError: invalid syntax

‚úÖ Fix: if x == True: or simply if x:

Misunderstanding operator precedence

    print("x is between 5 and 10")

‚ùå Error: SyntaxError: invalid syntax

‚úÖ Fix: if x > 5 and x < 10:

Capitalization matters

    print("Bring an umbrella!")

‚ùå Error: NameError: name 'true' is not defined

‚úÖ Fix: is_rainy = True (with capital T)

Random Module

The random module helps you generate random numbers and make random choices in Python.

First, you need to import it:

import random

Common Random Functions:

random.randint(a, b)

Returns a random integer from a to b (inclusive)

random.randint(1, 6)  # Random dice roll

random.choice(sequence)

Returns a random element from a sequence

random.choice(["rock", "paper", "scissors"])

random.random()

Returns a random float from 0.0 to 1.0

random.random()  # Might return 0.7219...

random.shuffle(list)

Shuffles the items in a list (in-place)

cards = [1, 2, 3, 4, 5]
random.shuffle(cards)

Common Errors

Forgetting to import the random module

dice_roll = random.randint(1, 6)
print("You rolled:", dice_roll)

‚ùå Error: NameError: name 'random' is not defined

‚úÖ Fix: Add import random at the top of your code

Incorrect function parameters

import random
dice_roll = random.randint(6)  # Missing the start parameter
print("You rolled:", dice_roll)

‚ùå Error: TypeError: randint() missing 1 required positional argument: 'b'

‚úÖ Fix: random.randint(1, 6) - Both parameters are required

Import Statements

Import statements let you use code from other modules or libraries in your program.

Different Ways to Import:

Import the entire module

import random

Use with: random.randint(1, 6)

Import specific functions

from random import randint, choice

Use with: randint(1, 6) (no 'random.' needed)

Import with a nickname (alias)

import random as rnd

Use with: rnd.randint(1, 6)

Import everything (not recommended)

from random import *

Use with: randint(1, 6) (but harder to tell where functions come from)

Common Errors

Module not found

import maths  # Incorrect name (should be 'math')

‚ùå Error: ModuleNotFoundError: No module named 'maths'

‚úÖ Fix: import math (check the correct module name)

Using a function without the module name

import math
print(sqrt(16))  # Missing 'math.' prefix

‚ùå Error: NameError: name 'sqrt' is not defined

‚úÖ Fix: print(math.sqrt(16)) or use from math import sqrt

Import should be at the top

print("Let's do some math!")
import math  # Import not at the top

‚ö†Ô∏è This works but is bad practice

‚úÖ Fix: Put all imports at the top of your file

For Loops

For loops let you repeat code a specific number of times or for each item in a collection.

Syntax Breakdown:

for variable in collection
The variable name for each item as we loop through the collection (list, range, etc.)
:
Required colon marks the beginning of a code block
code block
Indented code that runs for each item (use the tab key)

Common Ways to Use For Loops:

Loop through a range of numbers

for i in range(5):
    print(i)  # Prints 0, 1, 2, 3, 4

Loop through a list

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Loop with a start and end value

for i in range(2, 5):
    print(i)  # Prints 2, 3, 4

Loop with a step value

for i in range(0, 10, 2):
    print(i)  # Prints 0, 2, 4, 6, 8

Common Errors

Forgetting the colon

for i in range(5)
    print(i)

‚ùå Error: SyntaxError: invalid syntax

‚úÖ Fix: for i in range(5):

Incorrect indentation

for i in range(5):
print(i)

‚ùå Error: IndentationError: expected an indented block

‚úÖ Fix: Indent the code inside the loop with the tab key

range() parameters order

for i in range(10, 1):
    print(i)  # This won't print anything

‚ö†Ô∏è This won't give an error, but it won't run at all

‚úÖ Fix: For counting down, add a negative step: range(10, 1, -1)

Functions

Functions are reusable blocks of code that perform a specific task. They help you organize your code and avoid repetition.

Syntax Breakdown:

def function_name
Define a function with a descriptive name
(
Opening parenthesis for parameters
parameters
Optional inputs the function can use (multiple parameters separated by commas)
)
Closing parenthesis
:
Required colon marks the beginning of a function body
code block
Indented code that runs when the function is called (use the tab key)
return value
Optional: Send a result back from the function

Common Errors

Forgetting to call the function

def say_hello():
    print("Hello, world!")

# Nothing happens because the function isn't called

‚ö†Ô∏è No error, but nothing happens

‚úÖ Fix: say_hello() to call the function

Wrong number of arguments

def add(a, b):
    return a + b

result = add(5)  # Missing one argument

‚ùå Error: TypeError: add() missing 1 required positional argument: 'b'

‚úÖ Fix: result = add(5, 3)

Trying to use a value without returning it

def add(a, b):
    a + b  # Doesn't return the result

result = add(5, 3)
print(result)  # Prints None

‚ö†Ô∏è No error, but result will be None

‚úÖ Fix: return a + b

Turtle Graphics

The turtle module lets you create drawings and simple animations. It's like having a virtual turtle that carries a pen and follows your commands!

Getting Started:

import turtle

# Create a turtle
t = turtle.Turtle()

# Make the turtle draw a shape
t.forward(100)  # Move forward 100 pixels
t.right(90)     # Turn right 90 degrees
t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)

# Keep the window open
turtle.done()

Common Turtle Commands:

Moving Commands:

  • t.forward(distance) - Move forward
  • t.backward(distance) - Move backward
  • t.right(angle) - Turn right by angle degrees
  • t.left(angle) - Turn left by angle degrees
  • t.goto(x, y) - Move to coordinates

Pen Controls:

  • t.penup() - Stop drawing when moving
  • t.pendown() - Start drawing when moving
  • t.pensize(width) - Change line thickness
  • t.pencolor("red") - Change pen color
  • t.fillcolor("blue") - Set fill color

Fill Commands:

  • t.begin_fill() - Start filling a shape
  • t.end_fill() - Finish and fill the shape

Other Commands:

  • t.speed(speed) - Set drawing speed (1-10)
  • t.hideturtle() - Hide the turtle cursor
  • t.showturtle() - Show the turtle cursor
  • t.clear() - Clear the drawing
  • t.reset() - Clear and reset position

Common Errors

Forgetting to import turtle

t = turtle.Turtle()  # Missing import

‚ùå Error: NameError: name 'turtle' is not defined

‚úÖ Fix: import turtle at the top of your code

Forgetting turtle.done()

import turtle
t = turtle.Turtle()
t.forward(100)
# The window might close immediately

‚ö†Ô∏è The window might close too quickly

‚úÖ Fix: Add turtle.done() at the end

Missing begin_fill() or end_fill()

t.fillcolor("blue")
# Drawing commands here
t.end_fill()  # Missing begin_fill()

‚ö†Ô∏è The shape won't be filled

‚úÖ Fix: Always use begin_fill() before drawing a shape to fill

PSEUDOCODE GUIDE

1

Introduction to Pseudocode

Pseudocode shows the steps of our program using simple English-like statements. It helps us plan our code before we write it in a programming language.

Key Benefits:
  • Break down problems into clear steps
  • Plan your program's logic without syntax details
  • Catch logic errors early
  • Communicate your algorithm to others
Note: Good pseudocode should be easy for both programmers and non-programmers to understand.
2

Basic Pseudocode Rules

Keywords

KEYWORDS are written in CAPITAL LETTERS variables and values are in lowercase

KEYWORDS like BEGIN, END, IF, THEN are written in uppercase to stand out.

Indentation

IF score >= 70 THEN DISPLAY "You passed!" ENDIF

Statements inside blocks are indented to show they belong together.

Main Keywords

Program Structure
BEGIN program statements END

Every pseudocode algorithm starts with BEGIN and ends with END. These keywords signal where your program starts and finishes, creating clear boundaries for your code.

English to Pseudocode Translation

Common English phrases and their pseudocode counterparts:

Input Keywords

"Ask the user for..." → INPUT

"Get input from the user" → INPUT

"Request the user to enter..." → INPUT

"User should provide..." → INPUT

Output Keywords

"Display to the user..." → DISPLAY

"Print out..." → DISPLAY

"Show the result..." → DISPLAY

"Tell the user..." → DISPLAY

Calculation Keywords

"Calculate..." → CALCULATE

"Compute..." → CALCULATE

"Work out..." → CALCULATE

"Find the value of..." → CALCULATE

Decision Keywords

"If..." → IF ... THEN

"When..." → IF ... THEN

"Otherwise..." → ELSE

"Check if..." → IF ... THEN

Age Calculator Example

STEP-BY-STEP EXAMPLE

Write a program to calculate a person's age. Ask the user for their birth year and the current year, then calculate and display their age.

Step 1: Read & Understand the Problem

Reading the Problem

Write a program to calculate a person's age. Ask the user for their birth year and the current year, then calculate and display their age.

Planning the Solution

Our program will need to:

  1. Get the birth year as input
  2. Get the current year as input
  3. Calculate age (current year - birth year)
  4. Display the age to the user

My Thinking Process

When I first read a problem, I look for key actions that need to happen:

1

"Ask the user for their birth year and the current year" tells me I need to get input from the user

2

"calculate" tells me I need to perform a mathematical operation

3

"display their age" indicates I need to show output to the user

Step 2: Create Program Structure

Breaking Down the Structure

We need to create a program structure with a clear beginning and end.

Inside this structure, we'll add the steps needed to solve the age calculation problem.

Program Structure

BEGIN # We will add our code steps here END

My Thinking Process

Every pseudocode algorithm should have a clear structure:

1

I start with BEGIN to show where my program starts

2

I'll end with END to show where the program finishes

This creates the container for all the steps in my solution.

Step 3: Add Input Steps

Getting User Input

Ask the user for their birth year and the current year, then calculate and display their age.

We need to get two pieces of information from the user: birth year and current year.

Adding the Input Steps

BEGIN # Ask for birth year DISPLAY "Enter your birth year: " INPUT birth_year # Ask for current year DISPLAY "Enter the current year: " INPUT current_year END

My Thinking Process

For each input we need, I follow a two-step pattern:

1

First, use DISPLAY to show a prompt asking for the information

2

Then, use INPUT to collect and store what the user enters

3

I choose meaningful variable names: birth_year and current_year clearly describe what we're storing

Step 4: Add Calculation

Performing the Calculation

Ask the user for their birth year and the current year, then calculate and display their age.

We need to calculate the age by subtracting the birth year from the current year.

Adding the Calculation Step

BEGIN # Ask for birth year DISPLAY "Enter your birth year: " INPUT birth_year # Ask for current year DISPLAY "Enter the current year: " INPUT current_year # Calculate age CALCULATE age = current_year - birth_year END

My Thinking Process

When I need to perform a calculation:

1

I use the CALCULATE keyword to show I'm doing a mathematical operation

2

I create a variable age to store the result

3

To find the age, I subtract birth_year from current_year

This simple calculation works because age is the difference between the current year and the year you were born.

Step 5: Add Output

Displaying the Result

Ask the user for their birth year and the current year, then calculate and display their age.

We need to show the calculated age to the user with a clear message.

Adding the Output Step

BEGIN # Ask for birth year DISPLAY "Enter your birth year: " INPUT birth_year # Ask for current year DISPLAY "Enter the current year: " INPUT current_year # Calculate age CALCULATE age = current_year - birth_year # Display the result DISPLAY "You are " + age + " years old." END

My Thinking Process

For the final step, I need to show the result to the user:

1

I use DISPLAY to show information to the user

2

I combine text strings with the age variable to create a complete sentence

3

The "+" symbol joins the variable with the text strings

This makes the output user-friendly by presenting the result in a complete sentence rather than just showing a number.

Tip: This is a simplified example. In a real program, you might want to check if the birth year is after the current year, or account for birth months and days for more accurate age calculation.

Practice Problems

Try writing pseudocode for these problems:

  1. Temperature Converter: Write a program that converts temperature from Celsius to Fahrenheit. The formula is: F = (C × 9/5) + 32
  2. Grade Calculator: Write a program that asks for a student's score (0-100) and displays their letter grade (A: 90-100, B: 80-89, C: 70-79, D: 60-69, F: below 60)
  3. Simple Calculator: Write a program that asks the user for two numbers and an operation (+, -, *, /), then performs the calculation and displays the result
Reminder: Break down each problem step by step, identify inputs and outputs, and use the appropriate pseudocode commands.

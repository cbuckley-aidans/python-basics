Understanding Data Types

In Python, not all data is the same. Different types of data are stored and processed differently.

STRINGS (str)

Text data - words, sentences, or any characters in quotes

name = "Alice" message = "Hello, World!" age_text = "15" # This is text, not a number!

Used for: Names, messages, addresses

INTEGERS (int)

Whole numbers - no decimal points

score = 100 lives = 3 age_number = 15 # This is a number!

Used for: Counting, scoring, calculations

Key Insight: The input() function always returns a string, even if the user types numbers! We need to convert strings to integers if we want to do maths with them.

CHUNK 2: THE INPUT() FUNCTION

TEACHER CHUNK
10 mins

Getting Information from Users

The input() function is Python's way of asking questions and waiting for answers. When Python encounters input(), it pauses and waits for the user to type something and press Enter.

How input() Works

💬

The input() Process

1. Shows a prompt → 2. Waits for typing → 3. User presses Enter → 4. Returns the text as a string

Basic Input

name = input("What is your name? ") print("Hello, " + name)
What is your name? Alice Hello, Alice

Getting Numbers

age_text = input("How old are you? ") age = int(age_text) print("Next year you'll be", age + 1)
How old are you? 15 Next year you'll be 16

Converting Strings to Integers

The int() Function

String "42"

Text that looks like a number but can't be used in calculations

int() converts

Transforms the string into a real number

Integer 42

Now it can be used for maths operations!

Without Conversion

num1 = input("Enter a number: ") num2 = input("Enter another: ") print(num1 + num2)
Enter a number: 5 Enter another: 3 53 # Oops! It joined the text!

With Conversion

num1 = int(input("Enter a number: ")) num2 = int(input("Enter another: ")) print(num1 + num2)
Enter a number: 5 Enter another: 3 8 # Correct! It added the numbers!

Real-World Applications and Common Mistakes

Now that you understand input() and type conversion, let's explore practical applications and learn about common errors you might encounter.

1

Common Input Errors to Avoid

Type Conversion Error

age = int(input("Age: ")) # User types: "fifteen" # Error: invalid literal for int()

Problem: int() can't convert words to numbers.

Fix: Add instructions for users to enter digits only.

Forgetting the Prompt

name = input() # User sees: (blank line) # Confusing!

Problem: No prompt tells users what to enter.

Fix: Always include a clear prompt message.

name = input("Enter your name: ")

Browser Note: In this learning environment, input() creates popup boxes. In a real Python environment (like IDLE or terminal), input would appear directly in the console.

2

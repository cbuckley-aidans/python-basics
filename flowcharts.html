Standard Flowchart Symbols

Flowcharts use different shapes to show different parts of your code:

1

Basic Flowchart Symbols

Terminal (Oval/Rounded Rectangle)

Shows where your program starts and ends.

Process (Rectangle)

Shows math steps and changing variables.

Decision (Diamond)

Shows if statements where the program chooses between options.

Input/Output (Parallelogram)

Shows input() and print() commands.

Note: Arrows connect these symbols to show the order of steps.

2

Symbol Usage in Python

Process Symbol (Rectangle)

sum = x + y
MATH STEPS

This is what we use for math and changing variables.

Decision Symbol (Diamond)

if score >= 70:
DECIDE

This is what we use for if statements that make decisions.

Input/Output Symbol (Parallelogram)

name = input("Enter your name: ")

print("Hello, " + name)
INPUT/PRINT

This is what we use for input() and print() commands.

Start/End Symbol (Rounded Rectangle)

# Start of program
# ...
# End of program
START/END

This shows where our program starts and ends.

Basic If Statement Flowchart

Simple If Statement

Here's a simple IF statement in Python:

score = 85 if score >= 70: print("You passed!")

Here's how this looks as a flowchart:

If the score is 70 or higher, we print "You passed!" If not, we skip that step.

2

If-Else Statement

Now let's see an IF-ELSE statement:

score = 65 if score >= 70: print("You passed!") else: print("You failed.")

Here's the flowchart:

The diamond shape asks "is score >= 70?" If yes, go one way. If no, go the other way.

INTERMEDIATE FLOWCHARTS

INTERMEDIATE

If-Elif-Else Flowchart

1

Multiple Conditions

For more choices, we can use if-elif-else statements:

score = 85 if score >= 90: grade = "A" elif score >= 70: grade = "B" else: grade = "F" print("Your grade is: " + grade)

Here's the flowchart:

Each diamond asks a question. We keep checking until one answer is "yes", or we reach the end.

2

Simple Ticket Calculator

Here's a simple program that takes input and makes a decision:

def simple_ticket_calculator(): # Get user input age = int(input("Enter your age: ")) # Apply simple age-based pricing if age < 12: price = 5.00 # Child price else: price = 10.00 # Adult price # Output the result print(f"Your ticket price is: ${price:.2f}") # Call the function simple_ticket_calculator()

Notice how we use parallelograms for input() and print(), and a diamond for the if statement.

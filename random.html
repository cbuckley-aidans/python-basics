What Are Python Libraries?

1

Understanding Libraries

Python libraries are like toolboxes filled with useful tools (functions) that help us solve specific problems.

What is a Library?

A collection of pre-written code

  • Saves time - use code others have already written
  • Solves common problems without starting from scratch
  • Tested and reliable code you can trust
Examples of Popular Python Libraries

Math Library

Contains functions for mathematical operations

Random Library

Generates random numbers and makes random choices

Datetime Library

Works with dates and times

2

How to Import Libraries

Before we can use a library, we need to import it. There are different ways to import libraries in Python:

# Method 1: Import the entire library import random # Method 2: Import specific functions from random import randint # Method 3: Import with an alias import random as rnd

Import statements should always be at the beginning of your Python file.

Example: Using Method 1

import random random_number = random.randint(1, 10) print(random_number)

When using the full library name, we need to use the library name followed by a dot, then the function name.

Example: Using Method 2

from random import randint random_number = randint(1, 10) print(random_number)

When importing a specific function, we can use it directly without the library name.

3

What are Functions and Parameters?

Step 1: Understanding Functions

What is a Function?

A function is a block of code that performs a specific task.

Think of a function like a mini-program that does one job really well.

Real-World Example

A microwave has functions like "defrost" or "reheat" that perform specific tasks when you press a button.

Step 2: Understanding Parameters

What are Parameters?

Parameters are values that we pass to a function to customize how it works.

They're like the settings or inputs that the function needs to do its job.

Real-World Example

Setting the time on a microwave is like passing a parameter - it tells the microwave how long to run.

Step 3: Function Syntax
function_name(parameter1, parameter2)
function_name
The name of the function
+
(parameter1, parameter2)
Input values inside parentheses

Calling a function means using it in your code by writing the function name followed by parentheses with any required parameters.

Generating Random Numbers with randint()

1

Understanding randint()

The randint() function from the random library lets us generate a random integer between two values (inclusive).

from random import randint random_number = randint(minimum, maximum)

Parameters Explained

  • minimum: The smallest possible number that can be generated
  • maximum: The largest possible number that can be generated

Important: Both the minimum and maximum values are included in the possible results!

Example

from random import randint # Generate a random number between 1 and 6 dice_roll = randint(1, 6) print("You rolled:", dice_roll)

Possible results: 1, 2, 3, 4, 5, or 6

2

How randint() Works

randint() in Action Step 1 of 3
from random import randint

What's Happening

We're importing just the randint() function from the random library.

This is like taking just one tool out of a big toolbox, so we can use it easily.

from random import randint
random_number = randint(1, 10)

What's Happening

randint(1, 10)
7

The randint() function picks a random number between 1 and 10.

It's like rolling a 10-sided die - we don't know which number we'll get until we roll!

from random import randint random_number = randint(1, 10)
print("The random number is:", random_number)

What's Happening

random_number
7
The random number is: 7

We store the random value in a variable called random_number and then print it out.

Fix the Code

Each of these code examples has an error. Can you find and fix them?

Example 1:

import randint dice_roll = randint(1, 6) print("You rolled:", dice_roll)

Problem: We can't import randint directly, it's inside the random module.

from random import randint dice_roll = randint(1, 6) print("You rolled:", dice_roll)

Example 2:

from random import randint random_number = randint(10) print(random_number)

Problem: randint() requires two parameters (min and max), not just one.

from random import randint random_number = randint(1, 10) print(random_number)

Example 3:

import random random_number = randint(1, 100) print("Random number:", random_number)

Problem: When importing the whole random module, we need to use random.randint().

import random random_number = random.randint(1, 100) print("Random number:", random_number)

Pseudocode Conventions

Pseudocode follows certain rules to keep it clear and easy to understand:

1

Basic Pseudocode Rules

Keywords

KEYWORDS are written in CAPITAL LETTERS variables and values are in lowercase

KEYWORDS like BEGIN, END, IF, THEN are written in uppercase to stand out.

Indentation

IF score >= 70 THEN DISPLAY "You passed!" ENDIF

Statements inside blocks are indented to show they belong together.

Structure Boundaries

WHILE count < 10 statements ENDWHILE

The end of each structure is clearly marked with END keywords.

Clear Calculations

CALCULATE area = length * width

Calculations should show the full formula, not just the operation name.

Note: Good pseudocode should be easy for both programmers and non-programmers to understand.

2

Main Keywords in Pseudocode

Program Structure

BEGIN program statements END

Every pseudocode algorithm starts with BEGIN and ends with END.

Selection/Decision

IF score >= 70 THEN DISPLAY "You passed!" ELSE DISPLAY "You failed." ENDIF

Used for making decisions in your program.

Input and Output

INPUT username DISPLAY "Hello, " + username

Getting data from users and showing information.

Loops

WHILE counter < 10 DISPLAY counter SET counter = counter + 1 ENDWHILE FOR i = 1 TO 5 DISPLAY i ENDFOR

Used for repeating sections of code.

Basic If Statement

1

Simple If Statement

Here's a simple IF statement in Python:

score = 85 if score >= 70: print("You passed!")

Here's the pseudocode version:

BEGIN SET score = 85 IF score >= 70 THEN DISPLAY "You passed!" ENDIF END

The IF structure has a condition followed by THEN, indented statements, and ends with ENDIF.

2

If-Else Statement

Now let's see an IF-ELSE statement:

score = 65 if score >= 70: print("You passed!") else: print("You failed.")

Here's the pseudocode version:

BEGIN SET score = 65 IF score >= 70 THEN DISPLAY "You passed!" ELSE DISPLAY "You failed." ENDIF END

The ELSE keyword provides an alternative path when the condition is false.

INTERMEDIATE PSEUDOCODE

INTERMEDIATE

Multiple Conditions and Loops

1

If-Elif-Else in Pseudocode

For more choices, we can use if-elif-else statements:

score = 85 if score >= 90: grade = "A" elif score >= 70: grade = "B" else: grade = "F" print("Your grade is: " + grade)

Here's the pseudocode version:

BEGIN SET score = 85 IF score >= 90 THEN SET grade = "A" ELSE IF score >= 70 THEN SET grade = "B" ELSE SET grade = "F" ENDIF DISPLAY "Your grade is: " + grade END

We use ELSE IF for multiple conditions. Some pseudocode styles use ELSEIF instead.

2

Loop Structures

Pseudocode can show different types of loops:

# While Loop in Python count = 1 while count <= 5: print(count) count += 1 # For Loop in Python for i in range(1, 6): print(i)

Here's how to write loops in pseudocode:

# While Loop in Pseudocode BEGIN SET count = 1 WHILE count <= 5 DISPLAY count SET count = count + 1 ENDWHILE END # For Loop in Pseudocode BEGIN FOR i = 1 TO 5 DISPLAY i ENDFOR END

Loops in pseudocode have clear start and end keywords. FOR loops show the count variable and range.

Simple Program Example

Here's a complete pseudocode example for a simple calculator program:

BEGIN DISPLAY "Simple Calculator" DISPLAY "1. Add" DISPLAY "2. Subtract" DISPLAY "3. Multiply" DISPLAY "4. Divide" INPUT choice DISPLAY "Enter first number: " INPUT num1 DISPLAY "Enter second number: " INPUT num2 IF choice == 1 THEN CALCULATE result = num1 + num2 DISPLAY num1 + " + " + num2 + " = " + result ELSE IF choice == 2 THEN CALCULATE result = num1 - num2 DISPLAY num1 + " - " + num2 + " = " + result ELSE IF choice == 3 THEN CALCULATE result = num1 * num2 DISPLAY num1 + " * " + num2 + " = " + result ELSE IF choice == 4 THEN IF num2 == 0 THEN DISPLAY "Error: Cannot divide by zero" ELSE CALCULATE result = num1 / num2 DISPLAY num1 + " / " + num2 + " = " + result ENDIF ELSE DISPLAY "Invalid choice" ENDIF END

This example combines input, output, decisions, and calculations. Notice how we can nest IF statements inside each other.

WHAT IS PSEUDOCODE?

1

The Basics

Pseudocode is simply writing out the steps to solve a problem in plain English, before you write actual code. Think of it as making a recipe or writing instructions for someone else to follow.

It helps you:

  • Plan your solution before jumping into programming
  • Think through the logic step by step
  • Communicate your ideas to others clearly
  • Spot problems in your thinking early

Key Point: There's no "perfect" way to write pseudocode. As long as it's clear and logical, you're doing it right!

2

Converting Instructions to Pseudocode

Let's see how you turn everyday instructions into pseudocode:

Everyday Instructions

Making a peanut butter sandwich:

  1. Get two slices of bread, peanut butter, and jelly
  2. Spread peanut butter on one slice
  3. Spread jelly on the other slice
  4. Put the slices together

Same Thing as Pseudocode

BEGIN Get bread, peanut butter, and jelly Spread peanut butter on first slice Spread jelly on second slice Put slices together to make sandwich END

What's Different?

  • BEGIN and END: Shows exactly where the process starts and stops
  • Simple sentences: Each step is one clear action
  • Logical order: Steps flow in the right sequence
  • No ambiguity: Each instruction is specific
3

From Problem to Pseudocode

Here's how you'd approach a simple programming problem:

The Problem

Ask someone for their name and age, then tell them how old they'll be next year.

My thinking process:

  1. I need to get their name
  2. I need to get their age
  3. I need to add 1 to their age
  4. I need to tell them the result

My Pseudocode

BEGIN Ask user "What's your name?" Get their name Ask user "How old are you?" Get their age Turn their age into a number Calculate next_year_age = age + 1 Show "Hi " + name + "! Next year you'll be " + next_year_age END

TEMPERATURE CONVERTER

1

Problem Description

Ask the user for a temperature in Celsius and convert it to Fahrenheit. The formula is: Fahrenheit = (Celsius × 9/5) + 32

Enter temperature in Celsius: 25 Temperature in Fahrenheit: 77.0 Thank you for using the temperature converter!
2

Step by Step Solution

Temperature Converter Steps Step 1 of 4

What do I need to do?

The problem asks me to:

  • Get a temperature from the user
  • Convert it from Celsius to Fahrenheit
  • Show the result

I'll need to use the formula: F = (C × 9/5) + 32

My Plan

  1. Start the program
  2. Ask the user for input
  3. Convert the input to a number
  4. Calculate the Fahrenheit temperature
  5. Display the result
  6. End the program

My Thinking Process

1

I need to break this down into simple steps. The user gives me Celsius, I give them Fahrenheit.

2

I need to convert the input to a number because users type text, but I need to do math.

3

I'll use the formula exactly as given and show my work clearly.

Getting Input

I need to ask the user for their temperature and save it as a number I can work with.

Pseudocode So Far

BEGIN Ask user "Enter temperature in Celsius: " Get their answer Turn their answer into a number called celsius END

My Thinking Process

1

I always start with BEGIN to show where my program starts.

2

I use Ask user to show messages to the user, and Get to collect their input.

3

I convert to integer because I need to do math. User input is always text first!

Doing the Math

Now I'll use the formula to convert Celsius to Fahrenheit: F = (C × 9/5) + 32

Pseudocode So Far

BEGIN Ask user "Enter temperature in Celsius: " Get their answer Turn their answer into a number called celsius Calculate fahrenheit = (celsius * 9 / 5) + 32 END

My Thinking Process

1

I'm using Calculate to create a new variable called fahrenheit.

2

The formula needs brackets for order of operations - multiply and divide first, then add.

3

I'm using the celsius variable that I created in the previous step.

Showing the Results

Now I need to display the result clearly and end the program nicely.

Enter temperature in Celsius: 25
Temperature in Fahrenheit: 77.0
Thank you for using the temperature converter!

Complete Solution

BEGIN Ask user "Enter temperature in Celsius: " Get their answer Turn their answer into a number called celsius Calculate fahrenheit = (celsius * 9 / 5) + 32 Show "Temperature in Fahrenheit: " + fahrenheit Show "Thank you for using the temperature converter!" END

My Thinking Process

1

I use Show to display the result with a clear label so the user knows what the number means.

2

I add a thank you message to be polite and show the program is finished.

3

I always end with END to show where my program stops.

TRAVEL TIME CALCULATOR

1

Problem Description

Ask the user for a distance (in kilometers) and calculate how long it takes to drive at 60 km/h.

Enter distance in kilometers: 120 At 60 km/h, this journey will take 2.0 hours.
2

Step by Step Solution

Travel Time Calculator Steps Step 1 of 4

What do I need to do?

This problem asks me to:

  • Get a distance from the user
  • Calculate travel time at 60 km/h
  • Show the result

I need the formula: time = distance ÷ speed

My Plan

  1. Start the program
  2. Ask for the distance
  3. Convert input to integer
  4. Calculate time (distance ÷ 60)
  5. Display the result
  6. End the program

My Thinking Process

1

This is simpler than the temperature example because the speed is fixed at 60 km/h.

2

I only need one input from the user - the distance.

3

The math is distance ÷ 60 to get hours.

Getting the Distance

I need to ask the user how far they're traveling and convert it to a number.

Pseudocode So Far

BEGIN Ask user "Enter distance in kilometers: " Get their answer Turn their answer into a number called distance END

My Thinking Process

1

I start with BEGIN like always, to show where the program starts.

2

I ask for the distance in kilometers to be clear about what units I want.

3

I convert to INTEGER so I can do division math on it.

Calculating Time

I'll divide the distance by 60 to get the travel time in hours.

Pseudocode So Far

BEGIN Ask user "Enter distance in kilometers: " Get their answer Turn their answer into a number called distance Calculate time_hours = distance / 60 END

My Thinking Process

1

I'm using time = distance ÷ speed, and the speed is 60 km/h.

2

I name the variable time_hours to be clear about what it represents.

3

This calculation will give me a decimal number (like 2.5 hours).

Showing the Results

I'll display the travel time clearly and end the program.

Enter distance in kilometers: 120
At 60 km/h, this journey will take 2.0 hours.

Complete Solution

BEGIN Ask user "Enter distance in kilometers: " Get their answer Turn their answer into a number called distance Calculate time_hours = distance / 60 Show "At 60 km/h, this journey will take " + time_hours + " hours." END

My Thinking Process

1

I include "60 km/h" in the message so the user knows what speed I used.

2

I use the + to combine text and numbers in the output message.

3

I end with END to show the program is finished.

COIN FLIP GAME

1

Problem Description

Create a simple coin flip game. Ask the user to guess heads or tails, flip a coin randomly, and tell them if they won.

Enter your guess (heads or tails): heads The coin landed on: tails Sorry! Better luck next time!
2

Step by Step Solution

Coin Flip Game Steps Step 1 of 4

What do I need to do?

This problem asks me to:

  • Get the user's guess (heads or tails)
  • Generate a random coin flip
  • Compare the guess with the result
  • Tell them if they won or lost

My Plan

  1. Start the program
  2. Ask for their guess
  3. Generate random coin flip
  4. Compare guess with result
  5. Display appropriate message
  6. End the program

My Thinking Process

1

This introduces something new - randomness! I need to simulate flipping a coin.

2

I need IF statements to compare and decide what message to show.

3

The user input is text ("heads" or "tails"), not a number this time.

Getting the Guess

I need to ask the user for their guess and store it as text (not convert to integer this time).

Pseudocode So Far

BEGIN Ask user "Enter your guess (heads or tails): " Get user_guess END

My Thinking Process

1

I start with BEGIN like always.

2

I tell the user exactly what options they have: "heads or tails".

3

I use GET user_guess (no integer conversion) because I need the text.

Flipping the Coin

I need to randomly choose heads or tails. I'll generate a random number and use it to decide.

Pseudocode So Far

BEGIN Ask user "Enter your guess (heads or tails): " Get user_guess Pick a random number between 1 and 2 If random number = 1 then coin_result = "heads" Otherwise coin_result = "tails" END

My Thinking Process

1

I generate a random number between 1 and 2 to simulate a coin flip.

2

I use an IF-ELSE statement to decide: if 1 = heads, if 2 = tails.

3

I store the result in coin_result so I can use it later.

Checking the Result

Now I compare the user's guess with the coin flip result and show the appropriate message.

Enter your guess (heads or tails): heads
The coin landed on: tails
Sorry! Better luck next time!

Complete Solution

BEGIN Ask user "Enter your guess (heads or tails): " Get user_guess Pick a random number between 1 and 2 If random number = 1 then coin_result = "heads" Otherwise coin_result = "tails" Show "The coin landed on: " + coin_result If user_guess = coin_result then Show "Congratulations! You guessed correctly!" Otherwise Show "Sorry! Better luck next time!" END

My Thinking Process

1

First, I show what the coin landed on so the user knows the result.

2

Then I use another IF-ELSE to compare their guess with the coin result.

3

I end with END to finish the program.

NUMBER GUESSING GAME

1

Problem Description

Generate a random number between 1 and 5. Ask the user to guess the number and tell them if they got it right or wrong.

I'm thinking of a number between 1 and 5. Enter your guess: 3 The number was 2. You were wrong!
2

Step by Step Solution

Number Guessing Game Steps Step 1 of 4

What do I need to do?

This problem asks me to:

  • Generate a random number (1-5)
  • Ask the user to guess
  • Convert their guess to integer
  • Compare and show if they're right or wrong

My Plan

  1. Start the program
  2. Generate random number (1-5)
  3. Tell user the range
  4. Get their guess and convert to integer
  5. Compare and show result
  6. End the program

My Thinking Process

1

This is similar to the coin flip, but with numbers instead of heads/tails.

2

I need to convert the user's guess to integer so I can compare numbers.

3

I'll use a smaller range (1-5) to make it easier than 1-10.

Picking a Secret Number

I need to generate a random number between 1 and 5 and tell the user what range to guess from.

Pseudocode So Far

BEGIN Pick a random number between 1 and 5, call it secret_number Show "I'm thinking of a number between 1 and 5." END

My Thinking Process

1

I start with BEGIN and immediately generate the secret number.

2

I call it secret_number to remind me it's hidden from the user.

3

I tell the user the range so they know what numbers are valid.

Getting Their Guess

I need to ask for their guess and convert it to an integer so I can compare numbers.

Pseudocode So Far

BEGIN Pick a random number between 1 and 5, call it secret_number Show "I'm thinking of a number between 1 and 5." Ask user "Enter your guess: " Get their answer Turn their answer into a number called user_guess END

My Thinking Process

1

I ask for their guess with a simple prompt.

2

I convert to INTEGER because I need to compare numbers, not text.

3

Now I have both numbers: secret_number and user_guess.

Checking the Answer

I'll compare their guess with the secret number and show if they got it right or wrong.

I'm thinking of a number between 1 and 5.
Enter your guess: 3
The number was 2. You were wrong!

Complete Solution

BEGIN Pick a random number between 1 and 5, call it secret_number Show "I'm thinking of a number between 1 and 5." Ask user "Enter your guess: " Get their answer Turn their answer into a number called user_guess If user_guess = secret_number then Show "Correct! You guessed it!" Otherwise Show "The number was " + secret_number + ". You were wrong!" END

My Thinking Process

1

I use IF-ELSE to check if the guesses match.

2

If they're wrong, I show them what the number actually was.

3

I end with END to finish the program.

SIMPLE CALCULATOR

1

Problem Description

Ask the user for two numbers and add them together. Display the result clearly.

Enter first number: 15 Enter second number: 27 15 + 27 = 42
2

Step by Step Solution

Simple Calculator Steps Step 1 of 4

What do I need to do?

This problem asks me to:

  • Get the first number from user
  • Get the second number from user
  • Add them together
  • Display the result nicely

My Plan

  1. Start the program
  2. Ask for first number
  3. Ask for second number
  4. Convert both to integers
  5. Add them together
  6. Display the equation and answer
  7. End the program

My Thinking Process

1

This is straightforward - I need two inputs and one addition.

2

I need to convert both inputs to integers so I can do math.

3

I'll show the result as "15 + 27 = 42" to make it clear.

Getting First Number

I'll ask for the first number and convert it to an integer right away.

Pseudocode So Far

BEGIN Ask user "Enter first number: " Get their answer Turn their answer into a number called first_number END

My Thinking Process

1

I start with BEGIN as always.

2

I call it "first number" to be clear about which one I want.

3

I immediately convert to INTEGER for the math.

Getting Second Number and Adding

Now I'll get the second number and add them together.

Pseudocode So Far

BEGIN Ask user "Enter first number: " Get their answer Turn their answer into a number called first_number Ask user "Enter second number: " Get their answer Turn their answer into a number called second_number Calculate answer = first_number + second_number END

My Thinking Process

1

I follow the same pattern: DISPLAY, GET, CONVERT for the second number.

2

I use + to add the numbers and store the result in "answer".

3

Now I have all three values: first_number, second_number, and answer.

Showing the Calculation

I'll display the calculation in a nice format showing the equation and the answer.

Enter first number: 15
Enter second number: 27
15 + 27 = 42

Complete Solution

BEGIN Ask user "Enter first number: " Get their answer Turn their answer into a number called first_number Ask user "Enter second number: " Get their answer Turn their answer into a number called second_number Calculate answer = first_number + second_number Show first_number + " + " + second_number + " = " + answer END

My Thinking Process

1

I use + to combine text and numbers into one message.

2

The format "15 + 27 = 42" makes it clear what calculation was done.

3

I end with END to finish the program.

NUMBER COUNTER

1

Problem Description

Ask the user for a number and count from 1 up to that number, displaying each number.

Enter a number to count to: 5 Counting from 1 to 5: 1 2 3 4 5 Finished counting!
2

Step by Step Solution

Number Counter Steps Step 1 of 4

What do I need to do?

This problem asks me to:

  • Get a number from the user
  • Count from 1 up to that number
  • Display each number as I count
  • This needs a LOOP!

My Plan

  1. Start the program
  2. Ask for a number
  3. Convert to integer
  4. Use a FOR loop to count
  5. Display each number in the loop
  6. End the program

My Thinking Process

1

This introduces something new - a FOR loop! I need to repeat an action multiple times.

2

I need to count from 1 to whatever number they give me.

3

The loop will display each number as it counts.

Getting the Target Number

I need to ask the user what number to count up to and convert it to an integer.

Pseudocode So Far

BEGIN Ask user "Enter a number to count to: " Get their answer Turn their answer into a number called target_number END

My Thinking Process

1

I start with BEGIN as always.

2

I ask for "a number to count to" so it's clear what I want.

3

I call it target_number because that's what I'm counting toward.

Creating the Counting Loop

Now I'll use a FOR loop to count from 1 up to the target number.

Pseudocode So Far

BEGIN Ask user "Enter a number to count to: " Get their answer Turn their answer into a number called target_number Show "Counting from 1 to " + target_number + ":" For counter from 1 to target_number Show counter End for END

My Thinking Process

1

I tell the user what I'm about to do: "Counting from 1 to..."

2

I use For counter from 1 to target_number to set up the loop.

3

Inside the loop, I show counter to display each number.

4

I use End for to show where the loop ends.

Completing the Program

I'll add a final message to show the counting is complete.

Enter a number to count to: 5
Counting from 1 to 5:
1
2
3
4
5
Finished counting!

Complete Solution

BEGIN Ask user "Enter a number to count to: " Get their answer Turn their answer into a number called target_number Show "Counting from 1 to " + target_number + ":" For counter from 1 to target_number Show counter End for Show "Finished counting!" END

My Thinking Process

1

After the loop finishes, I add "Finished counting!" to show it's done.

2

The FOR loop automatically counts from 1 to target_number.

3

I end with END to finish the program.

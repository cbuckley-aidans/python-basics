WHAT ARE VARIABLES?

A variable is a named container that stores data in a computer program. Think of it like a labeled box where you can put information and retrieve it later by referring to its name.

apple

Store and remember data

Make code reusable

By storing values in variables, you can reuse them many times

Variables make code readable by using descriptive names

Change values during execution

Variables can be updated as your program runs

VARIABLE DATA TYPES

In Python, variables can store different types of data. Each type has its own characteristics and operations that can be performed on it.

Explore Data Types

Click the buttons below to see different data types:

Current Type:

Value:

Python Type:

int

Integers (int)

Whole numbers without decimal points.

age = 25

Operations: Addition, subtraction, multiplication, division, etc.

Floating Point (float)

Numbers with decimal points.

price = 19.99

Operations: Same as integers, plus precise decimal calculations.

Strings (str)

Text enclosed in quotes.

name = "Alice"

Operations: Concatenation, slicing, methods like upper(), lower(), etc.

Booleans (bool)

Either True or False.

is_active = True

Operations: Logical operations (and, or, not) and comparisons.

Lists

Ordered collections of items.

scores = [95, 87, 92]

Operations: Indexing, slicing, appending, sorting, etc.

Other Data Types

  • Dictionaries (dict): Key-value pairs
  • Tuples: Immutable ordered collections
  • Sets: Unordered collections of unique items
  • None: Represents the absence of a value

Checking Variable Types

You can discover a variable's type using the type() function:

age = 25
name = "Alice"
price = 19.99

print(type(age))    # 
print(type(name))   # 
print(type(price))  #

Python is a dynamically typed language, which means variable types are determined automatically based on the assigned value.

Watch Out!

Type Conversion Errors

Be careful when mixing different types in operations!

"age: " + 25 # Error! Can't add string and integer
"age: " + str(25) # Correct! Convert int to string first

Don't Forget String Quotes

Strings must always be enclosed in quotes!

name = Alice # Error! Python thinks Alice is a variable
name = "Alice" # Correct! Use quotes for text

VARIABLE OPERATIONS

Once you have variables, you can perform various operations on them to manipulate their values.

Arithmetic Operations

Mathematical operations on number variables.

x = 10
y = 5

addition = x + y      # 15
subtraction = x - y   # 5
multiply = x * y      # 50
division = x / y      # 2.0
exponent = x ** y     # 100000
remainder = x % y     # 0

String Operations

Working with text variables.

first = "Hello"
last = "World"

# Concatenation
full = first + " " + last  # "Hello World"

# Repetition
repeated = first * 3  # "HelloHelloHello"

# Methods
uppercase = first.upper()  # "HELLO"
length = len(first)  # 5

Compound Assignment Operators

These operators perform an operation and assign the result back to the variable in one step:

Numeric Examples

x = 10

x += 5    # Same as: x = x + 5 (x is now 15)
x -= 3    # Same as: x = x - 3 (x is now 12)
x *= 2    # Same as: x = x * 2 (x is now 24)
x /= 4    # Same as: x = x / 4 (x is now 6.0)

These operators are shortcuts that make your code more concise.

String Examples

message = "Hello"

# Add to the existing string
message += " there"   # message is now "Hello there"
message += "!"        # message is now "Hello there!"

The += operator is commonly used to build strings incrementally.

Type Conversion

You can convert between different data types using conversion functions:

String Number Conversion

# String to number
age_str = "25"
age_num = int(age_str)    # 25 (as integer)

price_str = "19.99"
price_num = float(price_str)  # 19.99 (as float)

# Number to string
count = 42
count_str = str(count)    # "42" (as string)

Between Numeric Types

# Integer to float
x = 10
x_float = float(x)    # 10.0

# Float to integer (truncates decimal)
y = 7.8
y_int = int(y)        # 7 (decimal part is removed)

# Round a float
z = 7.8
z_rounded = round(z)  # 8

Conversion Warnings

  • Not all strings can be converted to numbers (e.g., int("hello") will cause an error)
  • Converting from float to int truncates the decimal part without rounding
  • Boolean conversions: Any non-zero number or non-empty string converts to True

VARIABLE SYNTAX

Variable Declaration

Step-by-Step Syntax Breakdown

Step 0 of 6

We'll examine each element of a variable declaration:

Let's create a variable to store a player's score.

First, you choose a variable name:

player_scoreThe variable name is what you'll use to reference this data later. Use descriptive names that explain what the variable contains.

Next, we add the assignment operator (=):

player_score The equals sign (=) is the assignment operator. It tells Python to assign the value on the right to the variable on the left.=

Finally, we add the value we want to store:

player_score = 100The value is the data you want to store in the variable. It can be a number, string, boolean, or other data type.

The complete variable declaration:

player_score = 100The complete statement creates a variable named player_score and assigns it the value 100.

You can declare multiple variables in a single line:

x, y, z = 10, 20, 30This creates three variables (x, y, and z) with values 10, 20, and 30 respectively.

It's good practice to add comments explaining your variables:

player_score = 100 # Initial score for the playerComments (starting with #) help explain what a variable is for. They are ignored by Python when running the code.

Changing Variable Values

Step 0 of 4

Once a variable is created, you can change its value later in your program:

Let's update the player's score after winning a level.

First, we have our original variable:

player_score = 100This is the initial value of the variable when it was first created. # Some game code happens here...

To change the value, we simply assign a new value to the same variable name:

player_score = 100 # Some game code happens here... player_score = 150The same variable name now refers to a new value (150). The old value (100) is no longer accessible.

Often, you'll calculate the new value based on the current value:

player_score = 100 # Player scores 50 more points player_score = player_score + 50This takes the current value of player_score (100), adds 50, and stores the result (150) back in player_score.

For common operations, Python has shorthand operators:

player_score = 100 # These are equivalent: player_score = player_score + 50 player_score += 50The += operator is shorthand for "add and assign." Similar operators include -=, *=, /=, etc.

Working with Multiple Variables

Step 0 of 3

Python provides several ways to work with multiple variables efficiently:

Let's explore how to create and update multiple variables at once.

You can assign values to multiple variables in a single line:

# Initialize variables a = 5 b = 10 # In other languages, you need a temporary variable # temp = a # a = b # b = temp # In Python, you can swap in one line a, b = b, aThis swaps the values of a and b. After this line, a will be 10 and b will be 5. print(a) # 10 print(b) # 5

Variable Naming Rules

Valid Variable Names:

  • score - Simple, one word
  • player_score - Words separated by underscore
  • playerScore - Camel case (second word capitalized)
  • score1 - Letters and numbers
  • _hidden - Can start with underscore

Invalid Variable Names:

  • 123score - Cannot start with a number
  • player-score - Cannot use hyphens
  • if - Cannot use reserved keywords
  • player score - Cannot contain spaces
  • player@score - Cannot use special characters

Variable Naming Best Practices

Be Descriptive

x = 28 (Unclear what this represents)
age = 28 (Clear meaning)

Use Consistent Style

Snake Case (Python's preference):

player_score = 100

Camel Case (Alternative):

playerScore = 100

Avoid Reserved Words

Don't use Python keywords as variable names:

if = 5 (Error: 'if' is a Python keyword)

Other reserved words: for, while, class, return, True, None, etc.

Length and Readability

a = 3.14159 (Too short)
the_mathematical_constant_pi = 3.14159 (Too long)
pi_value = 3.14159 (Just right)

Common Variable Mistakes

Forgetting to Initialize

Variables must be assigned a value before they're used!

print(score) # Error if score hasn't been created
score = 0
print(score) # Correct: initialize first

Case Sensitivity

Variable names are case-sensitive!

name = "Alice"
print(Name) # Error: Name is not the same as name
name = "Alice"
print(name) # Correct: same case

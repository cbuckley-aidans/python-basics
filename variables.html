Variables: The Building Blocks of Programs

A variable is like a labelled box that stores information. Just as you might label a box "TOYS" or "BOOKS" to remember what's inside, in Python we give variables descriptive names to help us remember what data they contain.

1

Variable Metaphor

Imagine variables as labelled containers:

score

Stores game points (e.g., 5)

name

Stores player name (e.g., "Alex")

age

Stores player age (e.g., 12)

2

Basic Syntax

In Python, we create variables using this pattern:

variable_name = value

For example:

score = 5 name = "Alex" age = 12

The equals sign (=) assigns the value on the right to the variable name on the left.

IMPORTANT!

The = Symbol in Programming

In maths: equals is a comparison

x = 5 means "x equals 5" or "x is the same as 5"

In Python: = is an assignment operator

x = 5 means "store the value 5 in the variable named x"

Understanding Python Data Types

Not all data is the same! Python has different data types to handle different kinds of information. Understanding these types helps you choose the right way to store and work with your data.

1

The Four Main Data Types

STRING (str)

Text values in quotes

name = "Alex" message = 'Hello World'

Use for: names, messages, any text

INTEGER (int)

Whole numbers (no decimals)

age = 15 score = -42

Use for: counting, IDs, whole amounts

FLOAT (float)

Decimal numbers

height = 1.75 price = 9.99

Use for: measurements, money, percentages

BOOLEAN (bool)

True or False values

is_student = True has_passed = False

Use for: yes/no, on/off, conditions

2

Working with Different Types

String Operations

first_name = "Alex" last_name = "Smith" full_name = first_name + " " + last_name # Result: "Alex Smith"

Number Operations

score = 85 bonus = 15 total = score + bonus # Result: 100

Important: You can't mix incompatible types! For example, "Hello" + 5 will cause an error.

3

Boolean Logic

Booleans are often created by comparing values:

age = 16 score = 85 is_teenager = age >= 13 and age <= 19 # True has_passed = score >= 50 # True is_adult = age >= 18 # False print(f"Is teenager: {is_teenager}") print(f"Has passed: {has_passed}") print(f"Is adult: {is_adult}")

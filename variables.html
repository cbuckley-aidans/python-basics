WHAT IS A VARIABLE?

A variable is like a special container that can store information in your program. Think of it as a labeled box where you can put different things and find them later. Variables are one of the most important tools in programming!

VARIABLE TYPES

Variables can store different kinds of information. Just like containers can hold different items (like a water bottle for liquids or a box for solid objects), variables are designed to store specific types of data.

Numbers

Store numeric values like:

  • integer Whole numbers (42)
  • float Decimal numbers (3.14)
score = 10
temperature = 98.6

Strings

Store text values like:

  • Words or sentences
  • Names or messages
  • Any text in quotes
name = "Alex"
message = "Hello world!"

Booleans

Store true/false values:

  • Yes or no answers
  • On or off states
  • Condition results
is_game_over = False
player_has_key = True

Other Types

Lists

Store collections of items in order

fruits = ["apple", "banana", "cherry"]

Dictionaries

Store data in key-value pairs

person = {"name": "Alex", "age": 12}

Type Matters!

The type of a variable determines what you can do with it:

You can add numbers

x = 5
y = 3
sum = x + y # Result: 8

You can join strings

first = "Hello"
last = "World"
message = first + " " + last # "Hello World"

Watch Out!

You can't mix types without converting them first:

age = 12
message = "I am " + age # Error!
age = 12
message = "I am " + str(age) # Works! "I am 12"

USING VARIABLES

Creating Variables

The Basic Formula:

name = value
Variable names should be descriptive!

Variable Name = The Value You Want to Store

Examples:

age = 12
name = "Jamie"
is_student = True

Naming Rules:

  • Can use letters, numbers, and underscores
  • Must start with a letter or underscore
  • Case sensitive (score is different from Score)
  • Cannot use spaces or special characters
  • Cannot use reserved words like if, for, while

Good Names vs. Bad Names

Good ✓
  • player_score
  • user_name
  • is_game_over
Bad ✗
  • x
  • abc123
  • thisisareallylongvariablename

Using Variables

Changing Values

score = 0 # Start with zero
score = 10 # Change to 10
score = score + 5 # Add 5 more (now 15)

Variables can be updated throughout your program

Using in Expressions

width = 5
height = 10
area = width * height # Result: 50

Variables can be used in calculations and expressions

Putting It All Together

# Game score tracking
player_name = "Alex"
current_score = 0
level = 1
lives_remaining = 3
game_active = True

# Player scores 100 points
current_score = current_score + 100

# Player completes the level
level = level + 1

# Player loses a life
lives_remaining = lives_remaining - 1

# Check if game is over
if lives_remaining <= 0:
    game_active = False

Pro Tips!

Shorthand for Updating

score = score + 1
score += 1

Both do the same thing! The second way is just shorter.

Multiple Variables

x, y, z = 10, 20, 30

You can create multiple variables in one line!

REAL-WORLD EXAMPLES

Game Score Tracker

# Initial game stats
player_name = "Alex"
score = 0
high_score = 1500
lives = 3

# Player collects a coin
score += 100

# Player gets hit by an enemy
lives -= 1

# Check if it's a new high score
if score > high_score:
    high_score = score
    print("New high score: " + str(high_score))

# Check if game is over
if lives <= 0:
    print("Game Over!")

This example shows how variables would be used to keep track of a player's progress in a game.

Shopping Cart Calculator

# Shopping cart items
item1_name = "T-shirt"
item1_price = 15.99
item1_quantity = 2

item2_name = "Jeans"
item2_price = 29.99
item2_quantity = 1

# Calculate subtotal for each item
item1_total = item1_price * item1_quantity
item2_total = item2_price * item2_quantity

# Calculate the cart total
subtotal = item1_total + item2_total
tax_rate = 0.08 # 8% sales tax
tax_amount = subtotal * tax_rate
total = subtotal + tax_amount

# Display the total
print("Your total is: $" + str(round(total, 2)))

This example shows how variables help calculate prices in an online shopping cart.

Grade Calculator

# Test scores
test1 = 85
test2 = 92
test3 = 78

# Calculate the average
num_tests = 3
sum_scores = test1 + test2 + test3
average = sum_scores / num_tests

# Determine the letter grade
if average >= 90:
    letter_grade = "A"
elif average >= 80:
    letter_grade = "B"
elif average >= 70:
    letter_grade = "C"
elif average >= 60:
    letter_grade = "D"
else:
    letter_grade = "F"

# Display the results
print("Average score: " + str(average))
print("Letter grade: " + letter_grade)

This example shows how variables can help calculate and track test scores and grades.

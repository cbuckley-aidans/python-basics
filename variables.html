Variables: The Building Blocks of Programs

A variable is like a labelled box that stores information. Just as you might label a box "TOYS" or "BOOKS" to remember what's inside, in Python we give variables descriptive names to help us remember what data they contain.

1

Variable Metaphor

Imagine variables as labelled containers:

score

Stores game points (e.g., 5)

name

Stores player name (e.g., "Alex")

age

Stores player age (e.g., 12)

2

Basic Syntax

In Python, we create variables using this pattern:

variable_name = value

For example:

score = 5 name = "Alex" age = 12

The equals sign (=) assigns the value on the right to the variable name on the left.

IMPORTANT!

The = Symbol in Programming

In maths: equals is a comparison

x = 5 means "x equals 5" or "x is the same as 5"

In Python: = is an assignment operator

x = 5 means "store the value 5 in the variable named x"

Common Coding Mistakes

Common Error:

name = Jai

Problem: Missing quotes around a string value.

Fix: Always put string values in quotes.

name = "Jai"

Common Error:

print("I am " + age + " years old.")

Problem: You cannot add strings and numbers together in Python.

Fix: Either use commas in print() or convert to string.

print("I am", age, "years old.")

or

print("I am " + str(age) + " years old.")

Getting Input from Users

So far, we've created variables with values we already know. But often, we want our programs to get information from the user. This is where the input() function comes in!

1

Using input()

The input() function pauses your program and waits for the user to type something:

# Ask the user for their name name = input("What is your name? ") # Use the entered name in a greeting print("Hello, " + name + "!")

Program Output (if user types "Jamie"):

What is your name? Jamie
Hello, Jamie!

The input() function can include a prompt message that will be shown to the user.

2

How input() Works

Input Function in Action Step 1 of 3
user_age = input("How old are you? ")
# This displays the prompt "How old are you? " # And waits for the user to type something

Program Execution

How old are you? |

The program is waiting for the user to enter their age

user_age = input("How old are you? ")
# The user typed "12" and pressed Enter # This value is now stored in the user_age variable

Program Execution

How old are you? 12

user_age
"12"

Note: input() always stores values as strings!

user_age = input("How old are you? ")
print("Next year you will be", int(user_age) + 1, "years old")
# We convert the string "12" to the number 12 # Then add 1 to get 13

Program Execution

How old are you? 12

Next year you will be 13 years old

Key Point: We had to convert the string "12" to the number 12 using int() before we could add 1 to it!

3

Using F-Strings

Python provides a modern way to combine text and variables called f-strings:

Old Way (String Concatenation)

name = "Alex" age = 12 # Using + to join strings (concatenation) message = "Hello, " + name + "! You are " + str(age) + " years old."

Notice that we need to convert the number to a string using str(age) before we can add it to other strings.

New Way (F-Strings)

name = "Alex" age = 12 # Using f-strings (notice the f before the quote) message = f"Hello, {name}! You are {age} years old."

With f-strings, we just put variables inside {curly braces}. Python handles the conversion automatically!

F-string tips:

  • Start with the letter f before the opening quote
  • Put variables or expressions inside {} curly braces
  • Works with both single and double quotes: f'like this' or f"like this"
4

Strings vs Integers

It's important to understand the difference between text (strings) and numbers (integers):

STRINGS

  • Text values in quotes: "hello", '42'
  • Can be joined using +: "hello" + "world"
  • Cannot be used in math operations
  • The input() function always returns strings
a = "5" # String (not a number!) b = "10" # String (not a number!) result = a + b # "510" (joined as text)

INTEGERS

  • Numeric values without quotes: 42, -7
  • Can be used in calculations: 5 + 10
  • Cannot be directly combined with strings
  • Must convert strings to integers using int()
a = 5 # Integer (number) b = 10 # Integer (number) result = a + b # 15 (added as numbers)

Common Error: Type Mismatch

age = input("Enter your age: ") # User enters "14" next_year = age + 1 # ERROR! Cannot add 1 to a string

This code will cause a TypeError because we're trying to add a number to a string.

The Fix: Type Conversion

age = input("Enter your age: ") # User enters "14" age_as_number = int(age) # Convert "14" to 14 next_year = age_as_number + 1 # Now we can add 1, result is 15

By converting the string to an integer first with int(), we can perform the addition correctly.

Solving Problems with Variables

Variables become really powerful when we use them to solve problems. Let's explore how variables can be used in a simple calculation.

1

Calculating Area

Here's a program that calculates the area of a rectangle:

# Store the dimensions in variables length = 8 width = 3 # Calculate the area using the variables area = length * width # Display the result print("Area:", area) # Output: Area: 24

Rectangle Visualisation

width = 3
area = 24
length = 8
2

The Power of Variables

What makes this approach powerful:

REUSABILITY

We can change the values of length and width, and the program will calculate a new area

READABILITY

The variable names make it clear what each value represents in our calculation

FLEXIBILITY

We can use the variables in multiple calculations and formulas if needed

Key Point: By using variables instead of hardcoded values, we can easily modify the program to work with different inputs.

More Variable Use Cases

1

TEMPERATURE CONVERTER

# Convert Celsius to Fahrenheit celsius = 30 fahrenheit = (celsius * 9/5) + 32 print(fahrenheit) # 86.0
2

TAX CALCULATOR

# Calculate price after tax price = 50 tax_rate = 0.1 # 10% total = price + (price * tax_rate) print(total) # 55.0
3

GREETING GENERATOR

# Create personalised greeting name = "Alex" time = "morning" greeting = "Good " + time + ", " + name + "!" print(greeting) # Good morning, Alex!

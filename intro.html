What is Coding?

1

Introduction to Programming

Coding (or programming) is like writing a recipe that a computer can follow. Just as a recipe has specific steps to make a cake, code has specific instructions to make a computer do tasks.

What is Syntax?

Syntax is the set of rules for writing code correctly. Just like English has grammar rules (capital letters, full stops), programming languages have syntax rules.

If you don't follow the syntax rules, the computer won't understand your instructions!

Why Python?

Python is a programming language designed to be easy to read and write. It's like learning a simplified version of a foreign language that computers understand.

Python is often the first programming language people learn!

Your First Python Command

The print() function is like a messenger in Python. Its job is to display information on the screen so you can see what your program is doing.

1

Why Print is Important

See Results

Shows you what your code is doing

Give Messages

Communicate with users of your program

Find Problems

Help debug and test your code

2

Your First Print Statement

Here's the most famous first program in computer science:

print("Hello, World!")

Output:

Hello, World!

Did You Know? "Hello, World!" has been the traditional first program for decades. It shows that your programming environment is working correctly!

The Anatomy of a Print Statement

Every print statement follows a specific pattern. Let's break it down step by step to understand the essential components.

1

What Can You Print?

Text (Strings)

print("Hello, Python!")
Hello, Python!

Text must be in quotation marks

Numbers

print(42)
42

Numbers don't need quotation marks

2

Common Beginner Mistakes

❌ Forgetting Quotes
print(Hello)

Python thinks "Hello" is a variable name

❌ Missing Parentheses
print "Hello"

Python 3 requires parentheses for functions

Remember: Every print statement needs the function name, parentheses, and quotes around text!

Beyond Basic Printing

1

Printing Multiple Items

You can print multiple items in one statement by separating them with commas:

print("Name:", "Alice", "Age:", 15)
Name: Alice Age: 15

Notice: Python automatically adds spaces between the items when you use commas.

2

Creating New Lines

Use the special character \n to create new lines in your output:

Multiple Print Statements

print("First line") print("Second line")

Using \n Character

print("First line\nSecond line")
First line Second line

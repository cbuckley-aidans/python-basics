What is Programming?

1

Programming Basics

Programming is like giving instructions to a computer. The computer will follow your instructions exactly as written.

Python is a programming language - it's how we communicate with computers in a way that both humans and computers can understand.

Today, we'll learn our first Python command: print()

Key Point

The print() function is like a messenger - its job is to display messages on the screen.

It's usually the first command people learn in Python!

2

Your First Print Statement

Let's look at a simple print statement:

print("Hello, World!")

When you run this code, you'll see:

Hello, World!

The computer reads your instruction and displays the text "Hello, World!" on the screen.

3

Anatomy of a Print Statement

Let's break down the print statement into its parts:

print("Hello, World!")

print - Function Name

This is the name of the function - it tells Python what action to perform.

Think of it as a verb or command in a sentence.

( ) - Parentheses

These hold the information that the print function will use.

The content inside is called the "argument" or "parameter."

" " - Quotation Marks

These tell Python that the content inside is text (a "string").

Without quotes, Python would think it's a variable or code.

Hello, World! - Content

This is the actual message you want to display.

It can be any text you want to show on the screen.

4

Different Things to Print

You can print different types of information using the print function:

Text (Strings)

print("Hello, this is text!")
Hello, this is text!

Text must be in quotation marks

Numbers

print(42)
42

Numbers don't need quotation marks

Important: If you put numbers in quotes, Python treats them as text, not as actual numbers you can do math with.

Understanding Common Errors in Print Statements

1

Missing Quotation Marks

Error: Missing Quotation Marks

print(Hello)

Error Message: NameError: name 'Hello' is not defined

Problem: Without quotation marks, Python thinks "Hello" is a variable name, not text.

Fixed Version

print("Hello")

Solution: Add quotation marks around text to tell Python it's a string.

2

Missing Parentheses

Error: Missing Parentheses

print "Hello"

Error Message: SyntaxError: Missing parentheses in call to 'print'

Problem: The print function needs parentheses to work properly.

Fixed Version

print("Hello")

Solution: Always use parentheses with functions in Python.

3

Mismatched Quotation Marks

Error: Mismatched Quotation Marks

print("Hello))

Error Message: SyntaxError: EOL while scanning string literal

Problem: Opening quote mark doesn't have a matching closing quote mark.

Fixed Version

print("Hello")

Solution: Make sure each opening quote has a matching closing quote.

4

Spelling Errors

Error: Misspelled Function Name

pirnt("Hello")

Error Message: NameError: name 'pirnt' is not defined

Problem: Python doesn't recognize 'pirnt' as a function - spelling matters!

Fixed Version

print("Hello")

Solution: Double-check the spelling of function names.

Printing Multiple Lines and Creating ASCII Art

1

Printing Multiple Lines

You can print multiple lines in a few ways:

Method 1: Multiple Print Statements

print("First line") print("Second line") print("Third line")
First line Second line Third line

Method 2: Using \n (Newline Character)

print("First line\nSecond line\nThird line")
First line Second line Third line

The \n is a special character that creates a new line

Note: \n is called an "escape character" - it tells Python to do something special (in this case, create a new line).

2

Triple Quotes for Multi-line Strings

For multi-line text, you can also use triple quotes:

print("""This is line 1 This is line 2 This is line 3""")
This is line 1 This is line 2 This is line 3

Triple quotes let you write multi-line strings exactly as you want them to appear, with line breaks included.

3

Creating ASCII Art

You can create simple artwork using text characters:

print(""" /\\ / \\ / \\ / \\ /________\\ """)
/\ / \ / \ / \ /________\

This creates a simple triangle using text characters.

Tip: Notice that we need to use a double backslash (\\) to display a single backslash. This is because the backslash is a special character in Python.

4

More ASCII Art Examples

Simple Smiley Face

print(""" ___ / \\ | O O| | \\_/ | \\___/ """)
___ / \ | O O| | \_/ | \___/

Simple Heart

print(""" /\\ /\\ ( \\/ ) \\ / \\ / \\/ """)
/\ /\ ( \/ ) \ / \ / \/

WHAT ARE CONDITIONALS?

A conditional statement tells a computer to execute different code depending on whether a condition is true or false. Think of it as a fork in the road - your program has to decide which path to take!

Interactive Example
Number Checker

Current Value:

5

Result:

The number is positive!

Python Code:

num = 5

if num > 0:

    print("The number is positive!")

elif num < 0:

    print("The number is negative!")

else:

    print("The number is zero!")

COMPARISON OPERATORS

Comparison operators let you compare two values to see how they relate to each other. They are the building blocks of conditional statements.

Equal to

Not equal to

Greater than

<

Less than

print("You're a teenager!")

print("It's freezing!")

Greater/Less Equal

>=

Greater than or equal

<=

Less than or equal

print("You passed!")

print("Game over!")

How They Work

Comparison Results

Every comparison produces a Boolean result - either True or False.

True Examples
  • 5 > 2 → True
  • 10 == 10 → True
  • 7 <= 7 → True
  • "cat" != "dog" → True
False Examples
  • 5 < 2 → False
  • 10 == 5 → False
  • 7 >= 10 → False
  • "hello" == "Hello" → False

Watch Out!

Common Mistake: = vs ==

Don't confuse assignment (=) with comparison (==)!

if score = 100: # Wrong! This assigns 100 to score
if score == 100: # Correct! This checks if score equals 100

Case Sensitivity

String comparisons are case-sensitive!

"Python" == "python" # False

If you need to ignore case:

name.lower() == "python" # Converts to lowercase first

EXTENSION: COMBINING CONDITIONS

Logical operators let you combine multiple conditions together to create more complex decision-making in your programs.

and

Both conditions must be True for the result to be True.

if age >= 13 and age <= 19: print("You are a teenager")

True and True = True

True and False = False

False and True = False

False and False = False

or

At least one condition must be True for the result to be True.

if day == "Saturday" or day == "Sunday": print("It's the weekend!")

True or True = True

True or False = True

False or True = True

False or False = False

not

Reverses a Boolean value - True becomes False, and False becomes True.

if not is_game_over: print("Keep playing!")

not True = False

not False = True

Using Parentheses for Complex Conditions

Parentheses are crucial for creating clear, complex conditions and controlling the order of operations.

Without Parentheses

if age >= 13 and age <= 19 or has_parent_permission:
    print("You can play this game!")

This is ambiguous. Does the or apply to both conditions or just the second?

Python's default: and has higher precedence than or, so it evaluates (age >= 13 and age <= 19) or has_parent_permission.

With Parentheses

if (age >= 13 and age <= 19) or has_parent_permission:
    print("You can play this game!")

This is clear: either you're a teenager OR you have permission.

Always use parentheses when combining different logical operators to make your code clear and avoid bugs.

Multiple Levels of Nesting

if (temperature < 0 and not has_coat) or (temperature > 30 and not has_water):
    print("Weather conditions are dangerous!")
else:
    print("You're prepared for the weather.")

The parentheses make it clear that we're checking for two dangerous scenarios:

  1. Cold without a coat (temperature < 0 and not has_coat)
  2. Hot without water (temperature > 30 and not has_water)

IF STATEMENT SYNTAX

IF Statement

Step-by-Step Syntax Breakdown

Step 0 of 6

We'll examine each syntax element of a simple if statement:

Let's check if the player has enough points to level up.

First, we start with the if keyword:

ifThe "if" keyword starts the conditional statement. It must be lowercase.

Next, we add the condition that will be evaluated:

if player_score >= 100The condition is an expression that evaluates to True or False. Here we're checking if player_score is greater than or equal to 100.

The condition must be followed by a colon (:):

if player_score >= 100:The colon is required after the condition. It tells Python that a code block is coming next.

The code that should run if the condition is True must be indented:

if player_score >= 100: print("Level up!")Indentation (usually 4 spaces) is required for all code that should execute if the condition is True. Python uses indentation to determine code blocks.

You can include multiple statements in the indented block:

if player_score >= 100: print("Level up!") player_level += 1 player_health = 100All statements at the same indentation level are part of the same code block and will execute if the condition is True.

The complete if statement with all parts:

if player_score >= 100: print("Level up!") player_level += 1 player_health = 100 print("Game continues...") # This line is outside the if blockThe if statement checks the condition, and if True, executes the indented code. After the indented block, code continues at the previous indentation level.

ELIF Statement

Adding ELIF Statements

Step 0 of 4

Often we need to check multiple conditions in sequence. That's where elif (else if) comes in:

Let's categorize a player's score into different achievement levels.

The elif statement follows an if statement:

if score >= 90: print("You earned an A!") elif score >= 80:The "elif" keyword stands for "else if" and allows you to check another condition if the previous condition was False.

The elif follows the same syntax rules as if:

if score >= 90: print("You earned an A!") elif score >= 80: print("You earned a B!")Like if statements, elif needs a condition, a colon, and an indented code block that runs if its condition is True.

You can include multiple elif statements to check many conditions:

if score >= 90: print("You earned an A!") elif score >= 80: print("You earned a B!") elif score >= 70: print("You earned a C!") elif score >= 60: print("You earned a D!")You can have as many elif statements as you need. Python checks them in order, and executes the first one with a True condition.

Python checks conditions in sequence and executes only the first matching block:

if score >= 90: print("You earned an A!") elif score >= 80: print("You earned a B!") elif score >= 70: print("You earned a C!") elif score >= 60: print("You earned a D!")If score is 95, only the if block runs (printing "You earned an A!"). The elif statements are skipped even though some conditions might also be True.

ELSE Statement

Adding ELSE Statements

Step 0 of 3

The else statement provides a default action when no conditions match:

Let's complete our grading example with a default case for failing grades.

The else statement comes after if/elif blocks:

if score >= 90: print("You earned an A!") elif score >= 80: print("You earned a B!") elif score >= 70: print("You earned a C!") elif score >= 60: print("You earned a D!") else:The "else" keyword doesn't need a condition - it runs if all previous if/elif conditions were False.

Like if and elif, the else block must be indented:

if score >= 90: print("You earned an A!") elif score >= 80: print("You earned a B!") elif score >= 70: print("You earned a C!") elif score >= 60: print("You earned a D!") else: print("You failed the course.") print("Please schedule a meeting with your teacher.")The else block can contain multiple statements, all indented at the same level.

The complete if-elif-else structure:

if score >= 90: print("You earned an A!") elif score >= 80: print("You earned a B!") elif score >= 70: print("You earned a C!") elif score >= 60: print("You earned a D!") else: print("You failed the course.") print("Please schedule a meeting with your teacher.") print("End of grade report.")This forms a complete decision structure. Only one block (if, elif, or else) will run, based on which condition matches first. After the structure, code continues at the previous indentation level.

Complete If Statement Anatomy

if condition:
    # indented code block
    statement 1
    statement 2
elif another_condition:
    # another indented code block
    statement 3
    statement 4
else:
    # final indented code block
    statement 5
    statement 6

Each part of the if statement has a specific purpose and must follow exact syntax rules.

Critical Syntax Rules

The Colon is Required

if x > 10 # Error! Missing colon
if x > 10: # Correct

Indentation Must Be Consistent

if x > 10: print("Greater") print("Error") # Wrong indentation
if x > 10: print("Greater") print("Still in if block") # Correct

Case Sensitivity Matters

IF x > 10: # Error! Python keywords are lowercase
if x > 10: # Correct

Conditions Must Evaluate to Boolean

if score > 0: # Evaluates to True/False
if player_name: # Checks if not empty
